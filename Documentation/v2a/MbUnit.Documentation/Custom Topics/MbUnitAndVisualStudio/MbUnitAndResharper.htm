MbUnit And Resharper

Resharper (R#) is a popular add-in for Visual Studio from jetbrains.com that offers built-in code analysis, unit test running and many, many code refactorings at the touch of a right click or keyboard shortcut. By default, the unit test runner that it provides supports only tests written against the nUnit framework. However, in January 2007, Albert Weinert created a plug-in for Resharper v2.5 which adds MbUnit support into the R# test runner. JetBrains have since released R# v3.0. It still doesn’t support MbUnit out of the box, but Albert has confirmed he is working on an update to his plug-in for this new version. This page will be updated when a final version is completed.

More information on the Resharper test runner can be found
