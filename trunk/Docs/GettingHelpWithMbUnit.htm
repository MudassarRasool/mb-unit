Getting Help With MbUnit

If you’re having problems with MbUnit, either in its use or in coding against it, the online technical community has produced many blogposts and articles about it which you might want to search through to see if anyone has a problem similar to yours. For example, you might like to try the following searches

  • Google BlogSearch
  • Technorati
  • IceRocket
  • Digg
  • Delicious
  • Magnolia

If you can't find any help there, please post your queries to one of our Google Groups

  • MbUnit-User is our general discussion list
  • MbUnit-Dev is our developer’s query list

Note that both of these groups require you to join the group before you can post a message to the board.

Reporting A Bug

If you believe you've found a bug or an issue in MbUnit, please submit your bug report to us on Google Code. When you do report your bug, please remember to include the following:

  • A good title for the bug so we know roughly what the problem is when we read it
  • A list of steps to reproduce the error \ problem and a screenshot of it as well if it might help
  • A description of what you believe should be happening instead.

Microsoft blogged a nice, easy set of bug reporting best practices a while ago which you might like to read to get a better gist of it.
